# Algorithm Performance Evaluation

## Observations
* **Sorting:** Quick Sort was faster than Merge Sort for the average case.
* **Knapsack:** Greedy is better for fractional items, while DP is required for 0/1 Knapsack to get the optimal result.
* **TSP:** The Held-Karp (DP) approach is efficient for small inputs but grows exponentially.
